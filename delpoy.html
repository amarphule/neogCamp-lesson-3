How to deploy site on Netlify

Created on Sep 22, 2021

We are going to learn how to make our site live free using Netlify. Click the link and you will redirected to the official netlify website. Now there are few steps to follow:

Step 1:

When you click above link or visit official netlify website you will land on home page of netlify. First you need to signUp to make it work for signUp you need to fill signUp form details. Once you signUp, click signIn enter your userid and password.

Step 2:

After signIn you will get screen like below image. Since you are new user, you don't have any repositories. There are few tabs on screen let's not worry about it right now. Click on New site from Git. It will open next page.

Step 3:

Now you need to have repositories on either of platform Github, GitLab , Bitbucket to make your site live.

Step 4:

As my repositories are on github so I am going to click on Github button.

In this step after clicking Github you redirected to the new window where you need to give neccesory permissions to allow netlify to use repository.

Step 5:

Now you are quit near to make site live. Here you see your list of github repositories. In search box type repository name which you want to link to your site on Netlify and press enter. Now select the repository.

Step 6:

On selection of repository you get site setting page for your repository. In this setting page you can provide information like Owner name, Branch to deploy. And click on Deploy site button. It takes to few seconds to deploy your site.

Step 7:

Wow, Look your site is live now. click on link https://objective-wozniak-fc9e85.netlify.app. (In your case it is different) to see your first live site.

You can change site name by clicking Domain settings >> Custom domains >> Options(Edit site name)>>type new name>>save
